# LoginForm
Koa.js app with:
  1. Cookies
  2. JWT
  3. Sessions
  4. moment.js

## Features:
* Login functionality is imlemented using Javascript Web Token, which allows authorized users to see protected page for 30 seconds.
* Token is stored with cookie.
* Protected page shows you number of sessions.
